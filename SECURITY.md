# Segurança

Este projeto lida com permissões sensíveis e recursos críticos do Android.

## Uso autorizado

Use o PSBlock somente:

- Em dispositivo próprio.
- Em ambiente de estudo.
- Em laboratório autorizado.
- Em cenário com permissão explícita.

## Relato de vulnerabilidades

Se encontrar uma falha:

1. Não publique dados sensíveis.
2. Não compartilhe chaves, senhas, tokens ou informações privadas.
3. Abra uma issue com descrição responsável.
4. Inclua dispositivo, versão do Android, root e passos de reprodução quando possível.

## Escopo sensível

Áreas que exigem cuidado:

- Device Admin.
- Wipe remoto autorizado.
- Pânico remoto.
- App lock.
- Logs locais.
- Controles de câmera, microfone, localização e Bluetooth.
- Integrações root/fallback.
